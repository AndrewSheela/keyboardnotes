🎼 Lesson 11 - Other Chords

Learn more chords other than major and minor.


Other Chord Types

Expanding harmony using extended, suspended, and altered chords

Why More Chords?

Basic Major and Minor chords form the foundation, but music becomes richer when we add extra notes or change intervals.

These chords add color, tension, and movement to songs.

Extended Chords

These chords extend beyond basic triads.

Major 7th → 1–3–5–7
C Major 7 → C E G B

Dominant 7th → 1–3–5–♭7
C7 → C E G B♭

6th Chord → 1–3–5–6
C6 → C E G A

9th Chord → 1–3–5–♭7–9
C9 → C E G B♭ D

Suspended Chords (Sus)

Suspended chords remove the 3rd and replace it.

Sus2 → 1–2–5
C sus2 → C D G

Sus4 → 1–4–5
C sus4 → C F G

Suspended chords sound open and unresolved.

Altered Triads

Diminished → 1–♭3–♭5
C dim → C E♭ G♭

Augmented → 1–3–♯5
C aug → C E G♯

These chords create strong tension and motion.

Flat & Altered Notes

♭5 lowers the fifth (used in jazz & blues).
Example: C7♭5 → C E G♭ B♭

♭6 adds darker color.
Example: C♭6 → C E♭ G A♭

These chords are common in jazz, gospel, and cinematic music.

Summary

All advanced chords are created by adding, removing, or altering notes from the Major scale.

Understanding formulas is more important than memorizing names.
